# Mastermind (Number Guessing Game)
A terminal-based Python game where you try to crack a secret 4-digit number generated by the computer. After each guess, the game reveals which digits you got in the correct position.
 
---
 
## How to Play
1. Run the game - the computer secretly picks a **4-digit number**
2. Enter your guess and receive positional feedback
3. Correct digits show their value, wrong positions show **X**
4. Keep guessing until you crack the code!
---
 
## Features
- Computer randomly generates a 4-digit number each round
- Positional feedback after every guess (`7 X X 3` style display)
- Digit count feedback so you know how many positions you got right
- Tracks and displays your attempt count
- Special message if you guess correctly on the first try
- Input validation for non-numeric input and wrong digit count
- Play again option after each round
---
 
## Example Round
 
```
Player 2 is Computer.
X X X X
Enter a number guess containing 4 digits: 1400
1 4 X X
Not quite! You got 2 digit(s) correct!
 
Attempt Counter: 1
Enter a number guess containing 4 digits: 1423
1 4 X 3
Not quite! You got 3 digit(s) correct!
 
Attempt Counter: 2
Enter a number guess containing 4 digits: 1413
1 4 1 3
You've become a Mastermind! The number was correctly guessed!

```

## Getting Started
 
### Prerequisites
- Python 3 installed

```bash

cd mastermind-python

python mastermind.py
```
